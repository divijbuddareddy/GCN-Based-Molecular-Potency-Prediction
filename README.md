This project builds a QSAR (Quantitative Structure–Activity Relationship) model that predicts the pIC50 potency of small molecules directly from their SMILES strings using a Graph Convolutional Network (GCN) implemented with PyTorch Geometric. The dataset contains molecules with SMILES and experimental pIC50 values; after removing rows with missing pIC50 and invalid SMILES, each molecule is converted into a molecular graph using RDKit, where atoms are nodes, bonds are edges, and each atom is described by a feature vector that encodes its element type, degree, formal charge, and aromaticity. A GCN with three graph convolution layers, global mean pooling, and a small fully connected network is trained to regress the pIC50 value, and its performance is evaluated on a held‑out test set using standard regression metrics such as MAE, RMSE, R² and the percentage of molecules predicted within a specified error tolerance (for example ±0.5 pIC50). To show the benefit of graph‑based learning, a classical QSAR baseline is also implemented by computing 2048‑bit Morgan (ECFP) fingerprints from SMILES and training a Random Forest regressor on the same data split, and the GCN is observed to achieve lower prediction error and higher R² than this baseline, indicating that directly learning from molecular graphs captures structure–activity relationships more effectively. Finally, the best GCN model is saved and deployed as an interactive Gradio application on Hugging Face Spaces, where a user can enter a SMILES string in a web interface and immediately obtain the predicted pIC50 along with example inputs and simple interpretation guidelines, making the system easy to access and demonstrate in practice.
